# AI Culture Lab

Exploring how AI can create new forms of interaction for cultural learning, storytelling, and heritage experiences.

## Vision

Traditional culture is often presented as static information.
This project explores how AI can transform culture into interactive, personal, and participatory experiences.

## Experiments

- embodied learning

## Design Principles

- Interaction before information
- Culture should be experienced, not memorized
- AI should enhance curiosity, not replace exploration
- Human creativity remains central
